Firebase Auto Setup

ระบบทะเบียนคุมเบิกจ่ายร้านค้า — ตั้งค่า Firestore อัตโนมัติ

⚙️ ใส่ Config
👤 ข้อมูล Admin
🔥 สร้าง Firestore
✅ เสร็จสิ้น
1 วาง Firebase Config ของคุณ
ไปที่ Firebase Console → Project Settings (⚙️) → Your apps → คลิก </> Web → Register app แล้วคัดลอก firebaseConfig ทั้งหมดมาวางด้านล่าง
2 ข้อมูลผู้ดูแลระบบ (Admin)
ต้องสร้าง User ใน Firebase Authentication → Users → Add user ก่อน แล้วคัดลอก UID มาวางด้านล่าง

คัดลอกจาก Firebase Console → Authentication → Users → คอลัมน์ User UID

ตัวเลือกข้อมูลตัวอย่าง (Sample Data)
3 กำลังสร้าง Firestore...
🔥 เริ่มต้นการตั้งค่า Firebase...
กำลังดำเนินการ...

🎉 ตั้งค่า Firebase สำเร็จแล้ว!

Firestore ถูกสร้างและเติมข้อมูลเรียบร้อย
ตอนนี้สามารถล็อกอินเข้าระบบได้เลยครับ

ขั้นตอนสุดท้าย! ต้องเพิ่ม Domain ใน Firebase ก่อน:
Firebase Console → Authentication → Settings → Authorized domains
→ Add domain → ใส่ domain GitHub Pages ของคุณ เช่น username.github.io
เปิดหน้าหลัก (ผู้บริหาร) หน้าจัดการ (เจ้าหน้าที่)
